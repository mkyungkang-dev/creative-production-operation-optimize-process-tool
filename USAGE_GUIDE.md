## 📋 작업 관리

### 작업 목록 보기

1. 왼쪽 사이드바에서 **"작업 관리"** 클릭
2. 필터 사용하기:
   - **팀**: 생산팀/물류팀 (관리자만 선택 가능)
   - **상태**: 대기중/진행중/완료

### 새 작업 만들기

1. 작업 관리 페이지에서 **"새 작업"** 버튼 클릭
2. 정보 입력:
   - 작업명 (필수)
   - 설명
   - 팀 선택
   - 담당자 배정
   - 예상 완료일 (필수)
   - 우선순위 (1-10)
3. **"생성"** 버튼 클릭

### 작업 수정하기

1. 작업 카드 클릭하여 상세 보기
2. **"수정"** 버튼 클릭
3. 변경할 내용 수정:
   - 작업명, 설명
   - **상태 변경**: 대기중 → 진행중 → 완료
   - 우선순위
   - 예상 완료일
   - 실제 완료일 (완료 시)
4. **"저장"** 버튼 클릭

### 댓글 작성하기

1. 작업 상세 페이지에서 아래로 스크롤
2. 댓글 입력란에 내용 입력
3. **전송 버튼 (종이비행기 아이콘)** 클릭

## 🔔 알림 사용하기

### 알림 확인하기

1. 우측 상단 **종 아이콘** 클릭
2. 읽지 않은 알림은 파란색으로 표시
3. 알림 클릭 시:
   - 해당 작업으로 이동
   - 자동으로 읽음 처리

### 알림 종류

- 🟢 **작업 배정**: 새 작업이 나에게 할당됨
- 🔵 **작업 업데이트**: 내 작업의 상태가 변경됨
- 🟡 **마감일 경고**: 마감일이 다가오는 작업

### 알림 정리하기

- 개별 읽음: 알림 클릭
- 전체 읽음: 알림 패널 상단 **"모두 읽음으로 표시"** 클릭

## 📊 프로세스 다이어그램

### 워크플로우 시각화 보기

1. 왼쪽 사이드바에서 **"프로세스 다이어그램"** 클릭
2. 전체 프로세스를 한눈에 확인:
   - **생산팀 프로세스**: 원자재 구매 → 제조 → 품질 검사 → 포장
   - **물류팀 프로세스**: 창고 준비 → 재고 업데이트 → 배송 → 확인

### 프로세스 노드 이해하기

각 노드는 작업을 나타내며:
- **⏳ 대기중**: 아직 시작하지 않음
- **🔄 진행중**: 현재 작업 중
- **✅ 완료**: 작업 완료

노드를 클릭하면 작업 상세 정보를 볼 수 있습니다.

## 👥 사용자 관리 (관리자만)

### 새 사용자 추가하기

1. 왼쪽 사이드바에서 **"설정"** 클릭
2. 사용자 관리 섹션에서 **"추가"** 버튼 클릭
3. 정보 입력:
   - 이름
   - 이메일
   - 비밀번호
   - 역할 (생산팀/물류팀/관리자)
4. **"추가"** 버튼 클릭

### 사용자 목록 보기

설정 페이지에서 모든 사용자를 확인할 수 있습니다:
- 이름, 이메일
- 역할 (색상 뱃지)
- 가입 날짜

## 🔄 실시간 업데이트

시스템은 **5초마다 자동으로 업데이트**됩니다:
- 새로운 작업이 추가되면 자동으로 표시
- 다른 사용자가 작업을 수정하면 자동 반영
- 새 알림이 오면 종 아이콘에 숫자 표시

## 💡 유용한 팁

### 작업 상태 관리
1. 작업 시작 시: **대기중 → 진행중**으로 변경
2. 작업 완료 시: **진행중 → 완료**로 변경하고 실제 완료일 입력

### 우선순위 활용
- **9-10**: 긴급 (빨간색 플래그)
- **5-8**: 보통 (노란색 플래그)
- **1-4**: 낮음 (초록색 플래그)

### 필터로 효율적인 작업 관리
1. **내 작업만 보기**: 작업 관리에서 담당자로 본인 선택
2. **진행 중만 보기**: 상태 필터를 "진행중"으로 설정
3. **팀별 보기**: 팀 필터 사용 (관리자)

### 댓글로 소통하기
- 작업 진행 상황 공유
- 문제 발생 시 빠른 문의
- 완료 보고 및 피드백

## ❓ 자주 묻는 질문 (FAQ)

**Q: 비밀번호를 잊어버렸어요**
A: 관리자에게 연락하여 비밀번호 재설정을 요청하세요.

**Q: 작업이 자동으로 업데이트되나요?**
A: 네, 5초마다 자동으로 새로고침됩니다.

**Q: 작업을 삭제할 수 있나요?**
A: 관리자만 작업을 삭제할 수 있습니다.

**Q: 파일을 첨부할 수 있나요?**
A: 현재 버전에서는 지원하지 않습니다. 향후 업데이트 예정입니다.

**Q: 이메일 알림을 받을 수 있나요?**
A: 현재는 브라우저 내 알림만 지원합니다. 이메일 알림은 향후 추가될 예정입니다.
